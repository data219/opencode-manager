## Snippet A: Confluence wiki section lacked inline comment workflow

Before bulk/delete operations, always run a `search` with the exact same filter to verify scope.

## Confluence / Wiki Playbooks

### Page and space operations

```bash
atlcli wiki space list
atlcli wiki page list --space TEAM
atlcli wiki page get --id 12345
atlcli wiki search "architecture" --space TEAM
```

### Local docs sync lifecycle

```bash
atlcli wiki docs init ./docs --space TEAM
atlcli wiki docs pull ./docs
atlcli wiki docs status ./docs
atlcli wiki docs check ./docs --strict
atlcli wiki docs diff ./docs/page.md
atlcli wiki docs push ./docs --validate
```
